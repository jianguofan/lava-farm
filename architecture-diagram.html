数据流说明:
🔵 蓝色 — 群控桌面端: 用户操作入口, 多机管理/批量控制/实时监控
🟢 青色 — Broker 中间层: 统一接入, 消息路由, 认证鉴权, 状态缓存
🟣 紫色 — Moonraker: 每台打印机的 API Server, 对接 Klipper 固件
🟠 橙色 — MQTT: 轻量级发布/订阅, 设备状态上报 & 指令下发
🟢 绿色 — 硬件层: MCU 驱动步进电机/加热棒/传感器
⬛ 灰色 — 持久化: 历史数据存储 & 时序监控数据

核心链路: 桌面端 → Broker(路由/鉴权) → Moonraker(API) → MQTT(解耦) → Moonraker(订阅) → Klipper → MCU
