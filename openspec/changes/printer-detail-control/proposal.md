# Proposal: 单设备详情与控制

## Summary

增强单设备详情页，提供完整状态面板、温度曲线、远程控制、任务日志、耗材余量和摄像头占位/入口。

## Motivation

版本计划模块④要求完整设备监控面板和远程操作能力。当前已有 `PrinterDetailPage`、摄像头相关页面和服务，但需要按产品验收标准统一详情页结构。

## What It Does

- 展示设备基础信息、连接状态、打印进度。
- 展示喷嘴/热床温度和 30 分钟历史曲线。
- 提供暂停、继续、停止、设置温度、发送 G-code。
- 展示任务日志时间线。
- 展示耗材余量与低余量预警。
- 嵌入或跳转摄像头画面。

## Non-goals

- 不做复杂运动控制面板。
- 不实现远程公网访问。
- 摄像头阶段一允许占位或现有能力整合。

## Success Criteria

- 单设备详情中可完成常用远程操作。
- 远程操作响应 P99 ≤ 3 秒。
- 温度和任务日志可用于排查问题。
