# Proposal: 全屏控制面板

## Summary

将现有农场仪表盘升级为面向 100 台设备的全屏控制面板，包含统计栏、状态筛选、设备卡片网格、异常提示和右侧批量控制抽屉。

## Motivation

版本计划模块②要求一屏监控 100 台设备、状态刷新 ≤ 2 秒。参考图显示设备信息页需要突出总数、在线、运行中、空闲、完成，并通过卡片展示进度、温度、状态和异常。当前已有 `FarmDashboardPage`、`StatsBar`、`PrinterGrid`、`PrinterCard`、`BatchToolbar`，但批量控制仍是底部工具栏和弹窗，缺少右侧控制面板式确认流程。

## What It Does

- 主界面提供“设备信息”和“产品信息”导航。
- 设备信息页展示统计栏、筛选、设备卡片网格。
- 支持多选设备后打开右侧“批量控制”抽屉。
- 抽屉内选择操作类型、输入参数、查看已选设备表并提交。

## Non-goals

- 不在此 change 实现产品解析。
- 不替换底层 MQTT/HTTP 通信。
- 不实现云端远程控制。

## Success Criteria

- 100 台设备卡片首屏渲染 ≤ 3 秒。
- 状态刷新 P99 ≤ 2 秒。
- 批量操作提交后逐台结果可见，单台失败不阻塞整体。
